# Wetube
Cloning Youtube with Vanilla and NodeJS


## Installation

``` bash
$ npm install 
```

### Babel

``` bash
$ npm install @babel/node
$ npm install @babel/preset-env
$ npm install @babel/core
```

### Initialize

``` bash
npm install nodemon -D
```

``` bash
npm install morgan
```

``` bash
npm install helmet
```

``` bash
npm install body-parser
```

``` bash
npm install cookie-parser
```

``` bash
npm install pug
```
